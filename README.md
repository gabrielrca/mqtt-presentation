Como instalar o Mosquitto:

Pré requisitos:

Ubuntu Server 16.04 com acesso root 

Porta TCP:1883 aberta no seu firewall

Comandos

sudo apt-get update

sudo apt-get install mosquitto

sudo apt-get install mosquitto-clients
